jQuery中的DOM操作

发表于 2017-12-14 | 分类于 《锋利的jQuery》读书笔记

DOM是Document Object Model的缩写，即文档对象模型；DOM是一种与浏览器、平台、语言无关的接口，使用该接口可以轻松地访问页面中所有的标准组件，DOM操作分为：DOM Core、HTML-DOM和CSS-DOM：

  • DOM Core并不专属于JavaScript，任何一种支持DOM的程序设计语言都可以使用，其中有getElementById()、getElementsByTagName()、getAttribute()和setAttribute()等方法；
  • 在使用JavaScript和DOM为HTML文件编写脚本时，有许多专属于HTML-DOM的属性（如document.forms、element.src）；
  • CSS-DOM技术的主要作用是获取和设置style对象的各种属性；

1.查找节点

  • text()方法可以获取元素节点中的文本内容；
  • attr()方法可以获取元素节点的属性值，参数为要查询的属性的名字；

2.创建节点

$(html)方法会根据传入的HTML标记字符串，创建一个DOM对象，并将这个DOM对象包装成一个jQuery对象后返回。append()方法能将创建的新元素插入到文档中。

  • 动态创建的新元素节点不会被自动添加到文档中，需要其他方法插入；
  • 创建单个元素时，要注意闭合标签和使用标准的XHTML格式；

3.插入节点

  • append()：向每个匹配的元素内部追加内容，可一次追加多个内容，内容可以是DOM对象、HTML string、jQuery对象；
  • appendTo()：将所有匹配的元素追加到指定的元素中，$(A).appendTo(B)将A追加到B中；
  • prepend()：向每个匹配的元素内部前置内容，与append()类似；
  • prependTo()将所有匹配的元素前置到指定的元素中，与appendTo()类似；
  • after()：在每个匹配的元素之后插入内容；
  • insertAfter()：将所有匹配的元素插入到指定元素的后面，与appendTo()类似；
  • before()：在每个匹配的元素之前插入内容；
  • insertBefore()：将所有匹配的元素插入到指定的元素前面，与appendTo()类似；

4.删除节点

  • remove()方法删除所有匹配的元素，传入的参数用于根据jQuery表达式筛选元素，该节点包含的所有后代节点也将同时被删除，返回值是指向已被删除节点的引用；
  • datach()也是从DOM中去掉所有匹配的元素，但不会把匹配的元素从jQuery对象中删除，可以在将来再使用这些匹配的元素，与remove()不同，所有绑定的事件、附加的数据都会保留下来;
  • empty()清空元素中的所有后代节点；

5.复制节点

clone()方法可以复制元素节点（包括所有匹配元素、匹配元素的下级元素、文字节点），方法中传递一个参数true，表示可以在复制元素的同时复制元素中所绑定的事件；

6.替换节点

  • replaceWith()方法将所有匹配的元素都替换成指定的HTML或者DOM元素；
  • replaceAll()作用与replaceWith()相同，只不过颠倒了操作顺序；
  • 在替换之前为元素绑定的事件，在替换后会与被替换的元素一起消失，需要在新元素上重新绑定事件；

7.包裹节点

  • wrap()可以将某个节点用其他标记包裹起来，可以是selector、元素、HTML string和jQuery对象；
  • wrapAll()会将所有匹配的元素用一个元素来包裹，如果被包裹的多个元素间有其他元素，其他元素会被放到包裹元素之后；
  • wrapInner()会将每个匹配的元素的子内容（包括文本节点）用其他结构化的标记包裹起来；
  • unwrap()用于把包裹匹配元素的父级元素移除；

8.属性操作

  • attr()方法获取和设置元素属性，传递一个参数获取元素属性，传递两个参数设置元素属性，如果需要同时设置多个属性，可以采用“"attr":"value"”的形式；removeAttr()方法删除元素属性；
  • prop()和removeProp()分别用来获取在匹配元素集中的第一个元素的属性值和为匹配的元素删除设置的属性；

9.样式操作

  • attr()方法可以获取和修改元素的属性，同样可以获取和修改元素的class；
  • addClass()方法可以为元素在原有基础上添加class；
  • removeClass()可以从匹配的元素中删除全部或指定的class，可以以空格的方式移除多个class，当不带参数时就会将class的值全部移除；
  • toggle()方法可以控制行为上的重复切换，即交替执行多个函数；
  • toggleClass()可以控制样式上的重复切换，如果类名存在则删除它，如果类名不存在则添加它；
  • hasClass()用来判断元素中是否含有某个class，有返回true否则返回false；与is()方法等价；

10.设置和获取HTML、文本和值

  • html()方法类似于JavaScript中的innerHTML属性，可用来读取或者设置某个元素中的HTML内容；
  • text()方法类似于JavaScript中的innerText属性，可用来读取或者设置某个元素中的文本内容，innerText属性不适用于Firefox浏览器；
  • val()方法类似于JavaScript中的value属性，可用来设置和获取元素的值；如果元素为多选，则返回一个包含所有选择的值的数组；
  • focus()方法相当于JavaScript中的onfocus()方法，用来处理获得焦点时的事件；
  • blur()方法相当于JavaScript中的onblur()方法，用来处理失去焦点时的事件；

11.遍历节点

  • children()方法用于取得匹配元素的子元素集合，children()方法只考虑子元素而不考虑其他后代元素；
  • next()方法用于取得元素后面紧邻的同辈元素；
  • prev()方法用于取得匹配元素前面紧邻的同辈元素；
  • siblings()方法用于取得匹配元素前后所有的同辈元素；
  • closet()方法用于取得最近的匹配元素，如果当前元素匹配则直接返回元素本身，如果不匹配则逐级向上查找，否则返回一个空的jQuery对象；
  • parent()获得集合中每个匹配元素的父级元素；parents()获得集合中每个匹配元素的祖先元素；closest()从元素本身开始，主机箱上级元素匹配，并返回最先匹配的祖先元素；

12.CSS-DOM操作

CSS-DOM简单来说就是读取和设置style对象的各种属性：

  • css()方法可以获取元素的样式属性，也可以设置某个元素的单个样式，同时设置多个属性时需要采取“attribute:value”的形式；
  • height()方法取得匹配元素的当前高度值，也可以设置元素的高度（默认单位为px）
  • offset()获取元素在当前视窗的相对偏移，返回的对象包含top和left属性，且只对可见元素有效；
  • position()获取元素相对于最近的一个position样式属性设置为relative或absolute的祖先节点的相对偏移，返回的对象也包含top和left属性；
  • scrollTop()和scrollLeft()分别获取元素的滚动条距顶端的距离和距左侧的距离，也可以传递参数指定位置；
