JavaScript中的数据类型

发表于 2017-11-18 | 分类于 《JavaScript高级程序设计》读书笔记
  • 简单数据类型（基本数据类型）：Undefined、Null、Boolean、Number、String；
  • 复杂数据类型：Object；

1.typeof操作符

typeof操作符用于检测给定变量的数据类型：

  • undefined——未定义；
  • boolean——布尔值；
  • string——字符串；
  • number——数值；
  • object——对象或null；
  • function——函数；

typeof操作符的操作数可以是变量，也可以是数值字面量；也可以给操作数加上圆括号；

2.Undefined类型

Undefined类型只有特殊值undefined，使用var声明变量但未对其初始化时的变量值就是undefined；ECMA-262第3版引入该值是为了正式区分空对象指针与未经初始化的变量；
对于未声明过的变量，执行typeof操作符返回undefined值；对于未声明过的变量，只能使用typeof操作符检测其数据类型；

3.Null类型

Null类型只有特殊值null，逻辑角度来看null值表示一个空对象指针；若定义的变量将用于保存对象，最好将其初始化为null；undefined值派生自null值，因此ECMA-262规定对它们的相等性测试要返回true；

4.Boolean类型

Boolean类型只有字面值：true和false，转型函数Boolean()可以将其他类型值转换为对应的Boolean值：

数据类型 转换为true的值 转换为false的值
String 任何非空字符串 ""（空字符串）
Number 任何非零数字值（含无穷大） 0和NaN
Object 任何对象 null
Undefined n/a（不适用） undefined

5.Number类型

Number类型使用IEEE754格式来表示整数和浮点数值，八进制字面值（在严格模式下无效）以0开始，若字面值中的数值超出范围，则前导零将被忽略并以十进制数值解析，十六进制字面值以0x开始；

  • 浮点数值

保存浮点数值需要的内存空间是整数值的两倍，如果浮点数值本身表示的就是一个整数，该值会被转换为整数进行保存；

  • 数值范围

ECMAScript能表示的最小数值为Number.MIN_VALUE，即5e-324；最大数值为Number.MAX_VALUE，1.7976931348623157e+308；超过数值范围的值被自动转换成特殊的(-)Infinity值；访问Number.NEGATIVE_INFINITY和Number.POSITIVE_INFINITY也可得到相应Infinity的值；
isFinite()可以判断一个数值是否有穷；

  • NaN

NaN即非数值（Not a Number），用于表示一个本要返回数值的操作数却未返回数值的情况；任何涉及NaN的操作返回NaN；NaN与任何值都不相等，包括NaN本身；
isNaN()会确定某个参数是否“不是数值”；isNaN()在接收参数后，会尝试将这个值转换为数值，某些不是数值的值会直接转换为数值；

  • 数值转换

转型函数Number()可用于任何数据类型：

  • Boolean值：true和false分别被转换为1和0；
  • null值返回0，undefined返回NaN；
  • 若字符串中只包含数字（包括前面带正号或负号的情况），则转换为十进制数值；若包含有效的浮点格式，则转换为对应的浮点数值；若包含有效的十六进制格式，则转换为相同大小的十进制整数值；若字符串为空则转换为0；若包含除上述格式之外的字符，则转换为NaN；
  • 调用对象的valueOf()方法，然后按前面的规则转换返回的值；若转换结果是NaN，调用对象的toString()方法后，再次按前面的规则转换返回的字符串值；

parseInt()函数会忽略字符串前面的空格，直至找到第一个非空格字符；若第一个字符不是数字字符或负号则返回NaN；若第一个字符是数字字符，则继续解析直到遇到非数字字符；parseInt()能够识别出整数格式，若字符串以0x开头则将其当作十六进制整数，若字符串以0开头则将其当作八进制数解析；还可以为该函数提供第二个参数作为转换时使用的基数，如var num1=parseInt("AF",16);；

parseFloat()从第一个字符开始解析至无效的浮点数字字符为止，但该函数只解析十进制值；而且如果字符串包含的是一个可解析为整数的数，parseFloat()会返回整数；

6.String类型

String类型用于表示由零或多个16位Unicode字符组成的字符序列；

  • 字符字面量

\xnn——以十六进制代码nn表示的一个字符（其中n为0～F）；
\unnnn——以十六进制代码nnnn表示的一个Unicode字符（其中n为0～F）；
任何字符串的长度都可以通过访问其length属性取得，若字符串中包含双字节字符，则length属性可能不会精确地返回字符串中的字符数目；

  • 转换为字符串

数值、布尔值、对象和字符串值都有toString()方法（null和undefined没有该方法），在调用数值的toString()方法时，可传递参数：输出数值的基数；
转型函数String()能将任何类型的值转换为字符串，值null返回”null”，值undefined返回”undefined”；

7.Object类型

Object实例的属性和方法：

  • constructor：保存着用于创建当前对象的函数（构造函数）；
  • hasOwnProperty(propertyName)：检查给定属性在当前对象实例中是否存在，作为参数的属性名须以字符串形式指定；
  • isPrototypeOf(object)：检查传入的对象是否是传入对象的原型；
  • propertyIsEnumerable(propertyName)：检查给定属性是否能够用for-in语句来枚举，作为参数的属性名须以字符串形式指定；
  • toLocaleString()/toString()：返回对象的字符串表示；
  • valueOf()：返回对象的字符串、数值或布尔值表示；
