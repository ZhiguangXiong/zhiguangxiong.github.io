window对象

BOM的核心对象是window，表示浏览器的一个实例；在浏览器中，它既是通过JavaScript访问浏览器窗口的接口，又是ECMAScript规定的Global对象；在网页中定义的任何一个对象、变量和函数，都以window作为其Global对象；

1.全局作用域

  • 所有在全局作用域中声明的变量、函数都会变成window对象的属性和方法；
  • 使用var语句添加的window属性（即全局变量）的[[Configurable]]特性值为false，所以全局变量不能通过delete操作符删除，而直接在window对象上的定义的属性可以；
  • 尝试访问未声明的变量会抛出错误，但通过查询window对象，可以知道某个可能未声明的变量是否存在；

JavaScript中的函数

发表于 2017-11-25 | 分类于 《JavaScript高级程序设计》读书笔记

1.基础概念

  • 使用函数声明定义函数：function functionName(arg0, ···){···}；
  • Firefox/Safari/Chrome/Opera中，函数有非标准的name属性，可以访问到给函数指定的名字；
  • 函数声明提升是指在执行代码之前会先读取函数声明；
  • 使用函数表达式定义函数：var functionName = function(arg0, ···){···};；
  • 使用函数表达式创建的函数叫匿名函数，函数表达式在使用前必须先赋值，且匿名函数的name属性是空字符串；

  • 使用new操作符后跟Object构造函数：var person = new Object();；

JavaScript中的数据类型

发表于 2017-11-18 | 分类于 《JavaScript高级程序设计》读书笔记
  • 简单数据类型（基本数据类型）：Undefined、Null、Boolean、Number、String；
  • 复杂数据类型：Object；

1.typeof操作符

typeof操作符用于检测给定变量的数据类型：

  • undefined——未定义；
  • boolean——布尔值；
  • string——字符串；
  • number——数值；
  • object——对象或null；
  • function——函数；

typeof操作符的操作数可以是变量，也可以是数值字面量；也可以给操作数加上圆括号；
